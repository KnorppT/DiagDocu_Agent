# DiagDocu_Agent
Repo for an Agent which is able to generate the diagnostic documentation out of C and H-Files and the corresponding A2L.

So geht's:
1. DasAgent File diagdocu.agent.md herunterladen und im PST mit der Verzeichnisstruku .github/agents speichern.
2. In Visual Studio Code den PST über open folder hinzufügen.
3. Neuen Chat öffnen (oder den aktuellen Chat verwenden)
4. Oben im Chat-Eingabefeld auf das Agent-Dropdown klicken (das Symbol links neben dem Textfeld, oder einfach @ tippen)
"DiagDocu" aus der Liste auswählen

Deine Anfrage eingeben (Beispiel)
@DiagDocu Erstelle die DiagnoseDokumentation für DFC_rbe_CddUTnet_UTnet1Plaus
