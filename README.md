# DiagDocu_Agent
Repo for an Agent which is able to generate the diagnostic documentation out of C and H-Files and the corresponding A2L.

So geht's:
Neuen Chat öffnen (oder den aktuellen Chat verwenden)
Oben im Chat-Eingabefeld auf das Agent-Dropdown klicken (das Symbol links neben dem Textfeld, oder einfach @ tippen)
"DiagDocu" aus der Liste auswählen
Deine Anfrage eingeben,

Erstelle die DiagDocu für DFC_rbe_CddUTnet_UTnet1Plaus
